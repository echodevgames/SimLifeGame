## Overview

The game is structured as a hybrid between:
- Sandbox simulation
- Mission-driven progression

---

## Game Modes

### Story Mode
- Mission-driven progression
- Content locked behind objectives

### Free Mode
- All systems unlocked
- No progression restrictions

---

## Player Flow

1. Start with limited resources
2. Receive guided missions
3. Complete objectives
4. Unlock new systems/content
5. Expand gameplay options
