## Overview

The UI system communicates all core gameplay systems to the player.

It ensures that:

- The player always knows what to do next
- System states are visible and understandable
- Progression feels clear and rewarding

---

## Core UI Pillars

### 🎯 Clarity

- Information is easy to read and understand
- No ambiguity in objectives or system state

---

### 📋 Direction

- Missions provide constant guidance
- The player always has a clear next step

---

### ⚙️ System Feedback

- Systems respond visually to player actions
- Changes in efficiency, needs, and production are visible

---

### 📈 Progress Visibility

- Player growth is clearly shown
- Unlocks and upgrades feel meaningful

---

## Core UI Components

### 📋 Mission UI

- Displays active missions
- Tracks progress
- Highlights objectives

---

### ❤️ Needs UI

- Displays current needs (Hunger, Energy, Hygiene, Social)
- Provides warnings when needs are low

---

### 🌱 Production UI (CRITICAL)

- Displays current production status
- Shows output and efficiency
- Provides feedback on grow setups

---

## Design Goals

- Keep UI clean and readable
- Prioritize important information
- Reinforce gameplay systems through feedback
- Avoid overwhelming the player
