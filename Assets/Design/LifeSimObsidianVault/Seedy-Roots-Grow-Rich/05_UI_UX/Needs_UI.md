## Overview

The Needs UI displays the player’s current state and drives moment-to-moment decisions.

---

## Features

- Display core needs:
  - Hunger
  - Energy
  - Hygiene
  - Social

- Show current levels clearly
- Provide warnings for low or critical needs

---

## Behavior

- Updates in real-time
- Uses visual cues:
  - Color changes
  - Icons
  - Alerts

---

## Gameplay Role

- Encourages player to manage needs
- Creates pressure and decision-making
- Directly impacts grow system efficiency

---

## Design Goals

- Make needs instantly readable
- Provide clear urgency when needed
- Avoid clutter while maintaining visibility
