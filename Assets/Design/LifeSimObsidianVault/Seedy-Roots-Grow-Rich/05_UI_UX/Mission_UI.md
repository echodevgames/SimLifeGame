## Overview

The Mission UI provides the player with clear direction and progress tracking.

It is the most important UI element for guiding gameplay.

---

## Features

- Display multiple active missions
- Show objectives and progress
- Highlight key goals
- Provide completion feedback

---

## UI Elements

- Active mission list
- Progress bars / counters
- Objective descriptions
- Completion indicators

---

## Behavior

- Always visible or easily accessible
- Updates in real-time based on player actions
- Highlights newly assigned missions

---

## Design Goals

- Ensure the player always knows what to do next
- Make progression feel structured and rewarding
- Provide clear and immediate feedback
