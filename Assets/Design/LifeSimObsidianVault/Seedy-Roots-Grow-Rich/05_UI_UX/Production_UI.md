## Overview

The Production UI displays the status and performance of the player’s grow operation.

It is critical for understanding:

- Output
- Efficiency
- System performance

---

## Features

- Show active grow setups
- Display production progress
- Show output values
- Display efficiency indicators

---

## UI Elements

- Production timers / progress bars
- Output indicators (income or product)
- Efficiency metrics
- Setup status indicators

---

## Behavior

- Updates in real-time
- Reflects changes in setup, needs, and upgrades
- Provides clear feedback on system performance

---

## Gameplay Role

- Helps players optimize their setups
- Shows impact of upgrades and decisions
- Reinforces production as the core gameplay loop

---

## Design Goals

- Make production easy to understand at a glance
- Provide actionable feedback
- Reinforce the connection between systems and output
